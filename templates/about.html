A fun simple website made from FLASK! Here is are the technologies used to create this website:

TECHNOLOGIES:


Flask
Flask is a framework that handles server requests for your app - provides tools, libraries, and other technologies.

Python Script
Handles communication between webserver and the browser.

HTML
Responsible for the structure of the page content. It is the skeleton of the website.
- Stored in the 'templates' folder

CSS
Responsible for the styling & formatting of the HTML. It is the flesh of the website.
- Stored in the 'static' folder

SQLite
This will be our database system. We created a SQLite database from a CSV file - we used TablePlus to manage our data.

SQLAlchemy
This is our database of info for the webpage. It is the brain of the website.
- We will use ORMs (Object-relational mapping): gods created that so that instead of writing SQL, you can write python and the ORM will talk to the data-base for you - Create classes that model the DB itself

JINJA2
It is a templating language.
